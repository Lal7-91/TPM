DEDUPLICATION:
1. Compare titles and descriptions for overlap.
2. If an item is a subset of a package, exclude the standalone item.
3. Keep the more comprehensive option (packages > individual items).

SORTING:
Sort by relevance to user preferences (nature, old/historical, modern).
Tie-breakers: higher rating → higher reviews → lexicographically smaller title → preserve original order.

OUTPUT REQUIREMENTS:
Return a JSON list of top N items.
Each item must include all original fields (no "score" field).
If fewer than N valid items exist, return all valid items.
